## Design Context

### Users
Data engineers and developers evaluating Volley against Flink/Spark/Kafka Streams, and those ready to get started after hearing about it. They arrive with stream processing vocabulary — no hand-holding needed on concepts, but clear information architecture matters so both evaluators and onboarders find what they need fast.

### Brand Personality
**Calm, precise, confident.** Understated technical authority — lets the product speak. No hype, no overselling. The tone is factual and direct, like well-written documentation.

### Aesthetic Direction
- **Visual tone:** Minimal, restrained, technically polished. Glassmorphic cards with subtle depth. Blue-purple brand gradient used sparingly.
- **References:** Vercel/Next.js (glassmorphic, developer-focused, restrained animations), Tailwind CSS docs (warm technical clarity, excellent hierarchy), Apache Spark/Flink docs (content structure and depth for a processing framework).
- **Anti-references:** No enterprise SaaS energy (stock photos, "schedule a demo", corporate blue). No startup hype ("revolutionary", emoji-heavy copy, gratuitous gradients). Stay technical and grounded.
- **Theme:** System-aware light/dark via `prefers-color-scheme`. No toggle.
- **Typography:** Inter for prose, JetBrains Mono for code. Tight tracking on headings.
- **Motion:** Subtle fade-up reveals on scroll, hover micro-interactions (2-4px lifts). All disabled under `prefers-reduced-motion`.

### Design Principles
1. **Content over chrome** — Every visual element serves information hierarchy. No decoration for its own sake.
2. **Calm confidence** — The design should feel like a tool that knows it's good, without needing to shout. Understated > flashy.
3. **Developer-native** — Code samples, CLI commands, and technical specifics are first-class citizens, not afterthoughts.
4. **Restrained polish** — Glass effects, gradients, and animations are seasoning, not the main course. One accent per section max.
5. **Accessible by default** — Semantic HTML, focus-visible rings, contrast ratios, and reduced-motion support are non-negotiable foundations.
